# Aplikasi Input & Tampil Data (Tugas 6)

Project Flutter sederhana untuk tugas: menampilkan form input (Nama, NIM, Tahun Lahir) dan halaman hasil yang menampilkan informasi utama.

Fitur utama
- Halaman input dengan validasi untuk Nama, NIM, dan Tahun Lahir.
- Halaman hasil yang menampilkan Nama, NIM, dan umur (dihitung dari tahun lahir).

File penting yang diubah/ditambahkan
- lib/ui/form_data.dart — Form input, validasi, navigasi ke halaman hasil.
- lib/ui/tampil_data.dart — Halaman hasil yang menampilkan informasi utama (nama, NIM, umur).
- lib/main.dart — Mengubah halaman awal (home) menjadi FormDataPage dan mengatur warna tema.


Alur singkat penggunaan
1. Buka aplikasi — akan langsung muncul halaman "Input Data".
2. Isi Nama, NIM, dan Tahun Lahir lalu tekan tombol "Simpan".
3. Anda akan dibawa ke halaman hasil yang menampilkan Nama, NIM, dan Umur."
